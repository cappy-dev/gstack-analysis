TL;DR

gstack is a Claude Code skill suite by Garry Tan (YC CEO). It's designed for Claude Code specifically, not Hermes. Most skills use Claude Code's allowed-tools and hook system which Hermes doesn't support. However, several skills contain methodologies worth porting to Hermes skills:

  • investigate - systematic debugging methodology (4-phase root cause)
  • cso - security audit checklist (OWASP + STRIDE + secrets scanning)
  • review - pre-landing PR review checklist
  • ship - ship workflow (tests, diff review, version bump, changelog, PR)
  • scrape - web scraping pattern (prototype then codify)
  • design-review - visual QA for AI-generated designs (find AI slop patterns)
  • spec - turn vague intent into executable spec in 5 phases
